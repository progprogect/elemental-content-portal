# Elemental Content Creation Portal - Browser Extension

Browser Extension for integrating with Haygen AI Content Creator.

## Development

1. Install dependencies:
```bash
npm install
```

2. Build TypeScript:
```bash
npm run build
```

3. Load in Chrome:
   - Open `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select this `extension/` folder

## Building

```bash
npm run build
```

This compiles TypeScript files to JavaScript in the same directory structure.

## Icons

Place extension icons in `icons/` folder:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)
